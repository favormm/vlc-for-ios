Librairie de scan

Objectifs :

Permettre de récupérer tout plein de metadata sur un fichier vidéo. Une image, un titre, un sous titre, une année, etc... Ces informations devront être stockées par la librairie (elle gère donc le caching).

Parti pris On exclut les fichiers audio.

API

Voilà la liste des appels que l'on veut pouvoir faire :

GetInfoForFileName(fileName)

Appel bloquant qui retourne toutes les informations que l'on a sur le fichier 'filename'. Peut retourner "NULL" si l'on n'a pas encore indexé le fichier.

IndexContentOfFolder(folderPath)

Appel bloquant qui réindexe tout le contenu de 'folderPath'.

Architecture

On a une architecture en "pipeline" :

Etape 1 : Name cleaner

L'objectif est de nettoyer les noms de fichiers.

Cas vicieux

Les cas vicieux à prévoir (ou pas) :

  • Network share
  • Films dans un dossier (e.g. Dossier "99_Francs", contenant "Part_1.avi" et "Part_2.avi")
